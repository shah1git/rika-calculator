# RikaNV Lesnik 660L PRO — баллистический калькулятор

Реверс-инжиниринг + JavaScript-порт баллистического алгоритма из OEM-прошивки тепловизионного прицела RikaNV Lesnik 660L PRO (платформа SunCreative, Rockchip RK3576).

**Live demo:** [shah1git.github.io/rika-calculator](https://shah1git.github.io/rika-calculator/)

## Две версии

### 🔴 [calculator.html](./calculator.html) — Mirror прошивки
Байт-в-байт порт функции `ZS_TrajectoryCalculate` из `libGuiLesnik2_pro.so`, **включая найденные баги**. Используется для сверки расчётов с физическим прибором: если значения совпадают — баг существует, можно идти его исправлять.

### ✅ [calculator-corrected.html](./calculator-corrected.html) — С исправлениями
Та же модель, но с исправлениями четырёх багов баллистики прошивки. Добавлены поля длины и диаметра пули (которых нет в OEM-UI, что и есть один из багов).

## Что исправлено в corrected-версии

| Bug | Описание | Эффект для стрелка |
|---|---|---|
| 🔴 BUG-001 | Деривация (боковой снос) занижена в 6,45 раз: `/254` вместо `×0,0254` | На 1000 ярдов для .308 168 SMK: 16 см промах вправо |
| 🔴 BUG-006 | Стабильность пули жёстко считается под калибр .223 | Завышение SG на 22-37% для всех других калибров |
| 🟡 BUG-002 | Цикл интегрирования делает один лишний шаг после цели | ~1-2 см лишних к показанию падения |
| 🟢 BUG-008 | Расчёт пристрелки не использует динамический BC | Малый misalignment между расчётом пристрелки и траектории |

Подробное описание каждого бага простым языком — внутри `calculator-corrected.html` (раскрывающиеся карточки).

## Что входит в физическую модель

- Атмосфера ICAO ISA с учётом влажности (Magnus formula)
- Сопротивление воздуха: drag-таблицы G1/G2/G5/G6/G7/G8 (точные значения из прошивки)
- Коэффициент стабильности по Miller (с атмосферной поправкой)
- Боковой снос по формуле Litz (`(SG+1.2) · 1.25 · t^1.83`)
- Coriolis (`ω_ENU` × v, интегрирование на каждом шаге)
- Spin drift, поправка на наклон ствола, многосегментный ветер

## Что не включено

- Декомпили OEM-кода
- Asm-дампы
- Полное исследование багов (security findings, протокол debug-канала, и т.д.)

Это всё лежит в приватном репозитории по соображениям responsible disclosure.

## Использование

Открыть HTML-файл в браузере. Никаких зависимостей нет — drag-таблицы и константы зашиты в JS.

## Disclaimer

Это исследовательский проект. Не для использования в качестве боевого прицельного устройства. Точность баллистических расчётов зависит от точности входных данных (BC, MV, атмосферных параметров) и не гарантируется.

Цель

На этой дистанции рассчитываются падение, боковой снос, скорость и энергия. Чем дальше — тем заметнее Кориолис и спин-дрифт.
На ней траектория пересекает линию прицеливания (drop=0). Обычно 100 м. Угол пристрелки подбирается методом Ньютона.

Пуля

G1 — традиционная (плоское донце), G7 — современная boat-tail. Для match-пуль предпочитайте G7.
Берётся из таблиц производителя. Чем больше — меньше тормозит. У современных пуль публикуется как G1, так и G7.
Измеряется хронографом для конкретного оружия. Точность ±5 м/с критична: 10 м/с = ~5–10 см на 1000 м.
1 grain = 64.8 мг. Влияет на энергию (E=½mv²) и коэффициент стабильности SG.
.308=7.82, .338=8.59, .223=5.69, 6.5CM=6.71. Используется в формуле Miller stability.
Входит в Miller почти кубически — ошибка 10% по длине даёт ~30% по SG.
🔒 BUG-006: Прошивка игнорирует эти поля. Внутри жёстко прошиты значения для калибра .223 (диаметр 5,69 мм, длина 22,86 мм). Что бы вы тут ни ввели — расчёт стабильности будет считаться под .223. В corrected-версии эти поля работают.

Прицел

От центра окуляра до центра ствола. У типичной винтовки 30–50 мм. Заметнее на ближних дистанциях (до 200 м).

Атмосфера

Холодный воздух плотнее → больше сопротивление → пуля падает сильнее. Точность ±2°C важна для дальних.
1013 гПа = ст. на уровне моря. На высоте 1000 м ≈ 900 гПа. Метеостанция / барометр / GPS-альтиметр.
Парадоксально: влажный воздух менее плотный, чем сухой. Эффект на drop малый (~0.5%).

Ветер

Главный источник бокового сноса. На 1000 м боковой ветер 5 м/с даёт ~1 м сноса. Оцените на цели, не у стрелка.
12=в лицо, 6=в спину, 3=слева направо, 9=справа налево. Стрелок в центре циферблата лицом к цели.

Земля

Москва ≈ 55, Сочи ≈ 43, Мурманск ≈ 69. Положительная = северное полушарие. Влияет на Кориолис.
0=N, 90=E, 180=S, 270=W. Восток/запад дают максимум вертикальной составляющей Кориолиса (эффект Этвёша).

Ствол / нарезы

За сколько дюймов пуля делает один оборот. Меньше = быстрее. .308 = 1:10..1:12, .338 = 1:9..1:10.
Большинство стволов имеют правую нарезку. Определяет знак спин-дрифта: правая → пулю сносит вправо.

Коррекции

Вращение Земли. На 1000 м эффект 3–7 см. Включайте для дистанций >500 м. Spin drift (деривация) по формуле Litz. На 1000 м для .308 ≈ 19 см вправо при правой нарезке. Эффективный BC меняется на ±5–15% между сверхзвуком и дозвуком. Работает только для G1/G7.

Алгоритм — байт-в-байт порт ZS_TrajectoryCalculate из libGuiLesnik2_pro.so. Константы из бинаря: df_PIR=0.000208551, df_StandartDensity1=1.224998769, экспонента Litz 1.83, Ω=7.292115e-5 рад/с. Drag-таблицы G1/G2/G5/G6/G7/G8 с квадратичной интерполяцией (3 коэф./узел).

Что под капотом

Атмосфера: ρ = P_dry / (R_dry · T) + P_vap / (R_vap · T);   c = 331·√(T_K/273);   P_vap = Magnus(T_C)·RH
Сопротивление: a_drag = (ρ/ρ_std) · Cd(M) · |v_rel|² · v̂_rel / (BC · 1789.34)   — BC в lb/in², ρ_std=1.225 kg/m³
Кориолис: ω_ENU = (0, cos·Ω₀, sin·Ω₀), Ω₀=7.292115e-5 rad/s;   a_cor = −2·(ω × v_shoot);   интегрирование на каждом шаге
Деривация (Litz): drift = sign · (SG + 1.2) · 1.25 · t^1.83 [дюйм], где SG = Miller stability, t = TOF [сек], sign=+1 для правой нарезки
Miller stability: SG = 30·m_gr / (t_cal²·d_in³·L_cal·(1+L_cal²)) · (V/2800)^(1/3) · ((T_F+460)/519) · (29.92/P_inHg)

Алгоритм — порт ZS_TrajectoryCalculate из libGuiLesnik2_pro.so. Drag-таблицы (G1, G2, G5, G6, G7, G8) загружены из data/coefficient_table.bin, md5 которого совпадает у всех известных прошивок SunCreative. Шаг интегрирования: 0.5 м по дистанции, метод — explicit Euler. Zeroing — Newton-метод (макс 100 итераций, старт 2.5°).
